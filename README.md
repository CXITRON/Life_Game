## Getting Started

This project is a Flutter application. To run it locally:

1. Install the [Flutter SDK](https://flutter.dev/docs/get-started/install).
2. Install dependencies:

   ```bash
   flutter pub get
   ```

3. Launch the application:

   ```bash
   flutter run
   ```

4. Run the tests:

   ```bash
   flutter test
   ```

it finally works!
